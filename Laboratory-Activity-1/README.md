This activity focuses on using **digital signals** in Arduino.

### Key Features
- Uses pins **8 to 12**
- LEDs turn ON one by one (from 12 to 8)
- LEDs turn OFF one by one (from 12 to 8)
- Uses `delay(1000)` for 1-second intervals
