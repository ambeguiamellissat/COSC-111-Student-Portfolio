### Description
This activity connects Arduino to Python using `pyserial`.  
Python sends commands to control LEDs through serial communication.

### Key Features
- Uses pins **8 (Red)**, **9 (Green)**, **10 (Blue)**
- Serial commands:
  - `R/r` toggles Red LED
  - `G/g` toggles Green LED
  - `B/b` toggles Blue LED
  - `A/a` turns all ON
  - `0/o` turns all OFF
  - Others show error
- Python menu interface
